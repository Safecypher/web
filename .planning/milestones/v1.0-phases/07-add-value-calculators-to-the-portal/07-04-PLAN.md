    The file already has the `const BASE = process.env.NEXT_PUBLIC_SITE_URL ?? 'http://localhost:3000'` constant and the `'use server'` directive from Plan 03. Add a new exported function:

    ```typescript
    export async function fireMockupViewed(email?: string) {
      try {
        await fetch(`${BASE}/api/attio/event`, {
          method: 'POST',
          headers: {
            'Content-Type': 'application/json',
            'x-internal-token': process.env.INTERNAL_API_SECRET ?? '',
          },
          body: JSON.stringify({ event: 'mockup_viewed', email }),
        })
      } catch (err) {
        console.error('[Attio] fireMockupViewed failed', err)
      }
    }
    ```

    IMPORTANT: Use `${BASE}/api/attio/event` — NOT the relative URL '/api/attio/event'. Relative URLs are invalid in server-side fetch calls (no implicit host). Using a relative URL here will cause a TypeError in production.
